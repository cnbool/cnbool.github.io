    修改文件权限chmod
    三种权限:
    Read Write Execute
    1 1 1
    r=4
    w=2
    x=1
    rwx=7
    rw=6

    改变文件权限

    root的权限是 rwx（读写和执行权限），
    所属用户组权限是 rw (只有读写权限)，
    对于其它用户权限是 x(只有只执行权限)
